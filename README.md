# Search-Region-Pandas
Гео-классификатор, который каждой строке выставляет географическую принадлежность запроса определённому региону: если поисковый запрос содержит название города региона, то в столбце ‘region’ пишется название этого региона. Если поисковый запрос не содержит названия города, то ставится ‘undefined’.

Правила распределения по регионам Центр, Северо-Запад и Дальний Восток:
geo_data = {
‘Центр’: [‘москва’, ‘тула’, ‘ярославль’],
‘Северо-Запад’: [‘петербург’, ‘псков’, ‘мурманск’],
‘Дальний Восток’: [‘владивосток’, ‘сахалин’, ‘хабаровск’]
}
(в данной задаче классификация только по указанным регионам)

Результат классификации записывается в отдельный столбец region.
